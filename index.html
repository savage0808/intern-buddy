Connect to Firebase
One-time setup. Your Firebase config is saved in this browser only.
1. Go to console.firebase.google.com → create a project named intern-buddy
2. Build → Realtime Database → Create database → Start in test mode
3. Project Settings ⚙️ → Your apps → Add web app → copy values below
4. In Realtime Database → Rules: paste {"rules":{".read":"auth == null || auth != null",".write":"auth == null || auth != null"}} → Publish
Please fill in all three fields.
